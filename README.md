# employee-promotion-ml
Built a machine learning model to predict employee promotion. Performed EDA, handled missing values, encoded categorical data, and trained multiple models. Gradient Boosting gave the best performance and was used for final predictions.
# Employee Promotion Prediction (Machine Learning Project)

## 📌 Objective
The goal of this project is to predict whether an employee will be promoted based on various features such as education, experience, and performance.

---

## 📊 Dataset
The dataset is taken from Analytics Vidhya:
- Train dataset (used for training and evaluation)
- Test dataset (used only for prediction)
- Sample submission file

---

## 🔍 Steps Performed

### 1. Exploratory Data Analysis (EDA)
- Checked dataset structure and data types
- Identified missing values
- Analyzed target variable distribution

### 2. Data Preprocessing
- Handled missing values using mean imputation
- Encoded categorical variables using Label Encoding
- Ensured train and test data were processed separately

### 3. Model Building
Trained multiple machine learning models:
- Decision Tree Classifier
- Random Forest Classifier
- Gradient Boosting Classifier

### 4. Model Evaluation
- Compared models using accuracy score
- Selected the best-performing model

### 5. Final Model
- Gradient Boosting Classifier gave the best performance

---

## 📈 Result
- Best score achieved on Analytics Vidhya platform
- Predictions generated on test dataset and submitted
